# BTS Ping!
BTS Ping! é um bot para o Telegram e Twitter que tweeta e avisa em um canal especifico quando o BTS entrar em live no VLive!

## Como participar?
É só entrar no nosso canal do Telegram e/ou ativar as notificações do bot no Twitter.
São essas duas plataformas que o bot utiliza para avisar quando as lives são detectadas!
